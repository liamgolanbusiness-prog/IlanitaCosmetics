# ILANITA — קוסמטיקה פרא-רפואית | Para-Medical Cosmetics

Modern, mobile-first, RTL Hebrew website for **ILANITA — אילנית עמר**, a para-medical cosmetics studio in Ramat Gan with 30+ years of experience.

## Highlights

- 🇮🇱 Hebrew RTL layout
- 📱 Mobile-first, fully responsive
- ✨ GSAP + ScrollTrigger scroll animations
- 🌌 Three.js 3D animated hero (floating gold/rose particles + soft tori)
- 🎨 Custom luxe palette — pearl, deep rose, champagne gold, plum
- ♿ Accessibility — skip link, focus rings, aria, reduced-motion support
- 🔍 SEO — meta tags, OG/Twitter cards, schema.org `BeautySalon`, sitemap, robots.txt
- 💬 Floating WhatsApp button + click-to-call
- 🗺️ Embedded OpenStreetMap (no API key needed)

## Stack

Plain `HTML` + `CSS` + `JS` (no build step). External libs via CDN: GSAP, ScrollTrigger, Three.js. Fonts: Frank Ruhl Libre, Heebo, Cormorant Garamond.

## Run locally

Open `index.html` directly, or serve with any static server:

```bash
npx serve .
# or
python -m http.server 8080
```
